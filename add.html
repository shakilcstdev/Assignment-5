Question-1: What are the different ways to select an element in the DOM?

জাভাস্ক্রিপ্ট ব্যবহার করে DOM থেকে কোনো এলিমেন্ট নির্বাচন করার কিছু উপায় আছে। নিম্নলিখিত মেথডগুলো ব্যবহার করতে পারেন: getElementById – আইডি (id) অনুযায়ী একটি নির্দিষ্ট এলিমেন্ট নির্বাচন করতে। getElementsByClassName – নির্দিষ্ট ক্লাস নাম (class name) অনুযায়ী একাধিক এলিমেন্ট নির্বাচন করতে। getElementsByTagName – নির্দিষ্ট ট্যাগ নাম (tag name) অনুযায়ী একাধিক এলিমেন্ট নির্বাচন করতে। querySelector – CSS সিলেক্টরের মাধ্যমে প্রথম পাওয়া উপযুক্ত এলিমেন্ট নির্বাচন করতে। querySelectorAll – CSS সিলেক্টরের মাধ্যমে

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML, innerText, এবং textContent এর মধ্যে পার্থক্য হলো: innerHTML: এটি কোনো এলিমেন্টের সম্পূর্ণ HTML কন্টেন্ট নেয়, যার মধ্যে ট্যাগসহ সমস্ত HTML কোড অন্তর্ভুক্ত থাকে। innerText: এটি শুধুমাত্র দৃশ্যমান (ভিজিবল) টেক্সট রিটার্ন করে, অর্থাৎ যেসব টেক্সট ব্রাউজারে দেখা যায় সেগুলোই পাওয়া যাবে, কিন্তু লুকানো (hidden) টেক্সট আসবে না। textContent: এটি একটি এলিমেন্টের সব ধরনের টেক্সট রিটার্ন করে, যার মধ্যে হোয়াইটস্পেস এবং লাইন ব্রেকও অন্তর্ভুক্ত থাকে, তবে HTML ট্যাগগুলো এতে উপেক্ষা করা হয় (HTML ইন্টারপ্রেট হয় না)। সংক্ষেপে, innerHTML HTML কোডসহ পুরো কন্টেন্ট দেয়, innerText শুধু ভিজিবল টেক্সট দেয়, আর textContent এলিমেন্টের সব টেক্সট রিটার্ন করে, হোক তা দৃশ্যমান বা অদৃশ্য।

Question-3: What is event delegation in the DOM?

DOM-এ Event Delegation হলো একটি JavaScript টেকনিক, যেখানে আপনি একটি অভিভাবক (parent) এলিমেন্টে একটিমাত্র event listener সংযুক্ত করেন। এটি সেই অভিভাবক এলিমেন্টের অধীনে থাকা সন্তান (child) এলিমেন্টগুলোতে সংঘটিত ইভেন্টগুলোর হ্যান্ডলিং করতে পারে। এর ফলে প্রতিটি সন্তানে আলাদা আলাদা event listener যোগ করার প্রয়োজন হয় না।

Question-4: What is event bubbling in the DOM?

Event Bubbling হলো এমন একটি আচরণ যেখানে কোনো ইভেন্ট কোনো নির্দিষ্ট এলিমেন্টে ঘটলে, সেই এলিমেন্টের ইভেন্ট হ্যান্ডলার ট্রিগার হয় এবং একইসঙ্গে সেই ইভেন্টটি DOM (Document Object Model) এর উপরের দিকে (parent elements) ছড়িয়ে পড়ে। অর্থাৎ, ইভেন্টটি তার প্যারেন্ট এলিমেন্টগুলোর দিকেও যেতে থাকে যতক্ষণ না এটি ডকুমেন্টের মূল (root) অংশে পৌঁছে।

Question-5: How do you create, add, and remove elements using JavaScript?

আমি JavaScript-এ document.createElement() ব্যবহার করে নতুন এলিমেন্ট তৈরি করতে পারি, এবং তারপর appendChild() বা insertBefore() দিয়ে DOM-এ যোগ করতে পারি। কোনো এলিমেন্ট মুছতে চাইলে removeChild() বা parentNode.removeChild() ব্যবহার করে সেটার প্যারেন্ট এলিমেন্ট থেকে সরিয়ে ফেলতে পারি।
